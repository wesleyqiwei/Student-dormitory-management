## Student-dormitory-management 学生宿舍管理系统 Vue Element Node Koa Mysql


#### 欢迎交流需求，交流业务，交流技术 aya_xiyi@163.com


#### 前端-->web Setup

```sh
npm install
```

#### 前端-->web Start

```sh
npm run dev
```

#### 服务端-->server Start

```sh
node app.js
```


### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```

### 提交规范

 - feat 增加新功能
 - fix 修复问题/BUG
 - style 代码风格相关无影响运行结果的
 - perf 优化/性能提升
 - refactor 重构
 - revert 撤销修改
 - test 测试相关
 - docs 文档/注释
 - chore 依赖更新/脚手架配置修改等
 - workflow 工作流改进
 - ci 持续集成
 - types 类型定义文件更改
 - wip 开发中
 - build：主要目的是修改项目构建系统(例如 glup，webpack，rollup 的配置等)的提交
